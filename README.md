## CODE RUSH
### Description
CODE RUSH is typing game for practicing Linux commands.
It simulates an actual terminal.

**[LET'S PLAY, CLICK HERE.](https://ms16183.github.io/code-rush/)**

### Controls
- All Keys
- Mouse

### Build
- Godot v4.5 (GDScript)

```bash
git clone https://github.com/ms16183/code-rush.git
cd code-rush
# Export as HTML5
/path/to/godot --export "Web" build/index.html
```
